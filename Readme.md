## Установка

Для локальной установки и использования проекта вам потребуется Node.js v10 +.

```sh
npm i
```

## Использование

```sh
npm run start
```

### Программа обучения
1. SPA - Single Page Application
2. Структуры данных
3. ООП, введение.
4. ООП, наследование и полиморфизм.
5. MVP, Presenter.
6. MVP, View.
7. MVP, Model.
8. Работа с сетью.
9. Offline.
